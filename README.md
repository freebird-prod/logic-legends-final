Smart Customer Support Automation

This project is a full-stack customer support system that automates complaint handling using AI, Flask backend, and React frontend.

Key Features

1.Flask backend for API handling and routing
2.React frontend for complaint submission
3.AI integration to detect duplicates, classify priority, categorize issues, and generate responses
4.Automated email response simulation for moderate-priority issues
5.Extendable for Jira or other ticketing integrations

Tech Stack

1.Frontend: React + TypeScript
2.Backend: Flask (Python)
3.AI: Google Gemini API
4.Tools: Flask-CORS, Requests
